  • SBAScheduleManager

    Default data source handler for scheduled activities. This manager is used to get SBBScheduledActivity objects and upload the task results for Bridge services. By default, this manager will fetch all the activities, but will not cache them all in memory. Instead, it will filter out those activites that are valid for today and the most recent finished activity (if any) for each activity identifier where the activity identifier refers to an SBBActivity object’s associated SBAActivityReference.
