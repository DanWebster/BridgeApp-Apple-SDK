SBAModuleIdentifier

public struct SBAModuleIdentifier : RawRepresentable, Codable

RSDModuleIdentifier is intended to allow a developer to add a light-weight reference to active task modules (such as a tapping task or hole-in-peg test) by referencing an identifier string that can be used to reference a module defined in a different framework.
