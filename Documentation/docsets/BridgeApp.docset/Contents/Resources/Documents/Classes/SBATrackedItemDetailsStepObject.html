SBATrackedItemDetailsStepObject

open class SBATrackedItemDetailsStepObject : RSDFormUIStepObject, SBATrackedItemDetailsStep

SBATrackedItemDetailsStepObject is a concrete implementation of the SBATrackedItemDetailsStep protocol that includes a template for creating multiple schedules assigned to this tracked item details.
