SBASurveyConfiguration

open class SBASurveyConfiguration

SBASurveyConfiguration is a survey wrapper that can extend the default implementation of UI/UX handling for all the Bridge surveys used by a given app.

  • shared

    The shared singleton.

  • stepType(for:)

    Allow customization of the step type for a given survey element.

    Default implementation will check the stepTypeMap for a guid and return that value if found. Otherwise it will return .form for SBBSurveyQuestion classes and return .instruction for all other classes.

    The step type is used within the UI to determine the default view controller to associate with a given step.
