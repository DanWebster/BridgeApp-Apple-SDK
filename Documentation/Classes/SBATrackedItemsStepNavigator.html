SBATrackedItemsStepNavigator

open class SBATrackedItemsStepNavigator : Decodable, RSDStepNavigator

SBATrackedItemsStepNavigator is a general-purpose navigator designed to be used for selecting tracked data such as medication, triggers, or symptoms.
